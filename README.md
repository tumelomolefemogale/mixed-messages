# Mixed Messages

A simple JavaScript app that generates a random funny sentence every time you click a button.

## Features
- Randomly combines days, moods, and activities.
- Button interaction updates the message dynamically.
- Clean and simple CSS design.

## How to Use
1. Open the `index.html` file in your browser.
2. Click the button to generate a new random message.

## Built With
- HTML
- CSS
- JavaScript
